# Argo CD Bootstrap

Use this area for cluster-level Argo CD bootstrap manifests if you want to keep them in the GitOps repo.

Examples:

- Argo CD `AppProject`
- root `Application`
- namespace bootstrap applications

The per-project applications generated by Jenkins are written under:

- `apps/<workspace-id>/<user-id>/<project-name>/application.yaml`

So this folder is only for shared Argo CD bootstrap resources, not tenant app instances.
